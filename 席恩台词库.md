# 席恩 · 桌球对局语音触发表

> 角色：席恩（执事 / 硅基生命体）· 基调：执事壳 + 腹黑点缀 · 称呼：「您」
> 规模：11 类，共 **29 条**（每条均有真人语音 mp3）
> 出处：`game_final.js → VOICE_CLIPS.xien`（语音）/ `VOICE_LINES.xien`（文字兜底）
> 音频目录：`assets/voice/xien/*.mp3`
> 触发判定：`resolveShot()` 在每杆球停稳后，按「谁出杆 + 什么结果」选分类，冷却 9s，随机取一条

---

## 触发总览

| 何时 | 结果 | 分类 | 概率 |
|------|------|------|------|
| **玩家出杆** | 进球（单颗） | `praise` | 60% |
| | 连续进球（≥2 颗） | `praise_streak` | 70% |
| | 打丢没进 | `taunt` | 40%（没中则 25% 补 `chat`） |
| | 空杆 | `pfoul_empty` | 100% |
| | 其它犯规（碰错球 / 母球落袋） | `pfoul_other` | 100% |
| **席恩出杆** | 进球（含连击） | `brag` | 60% |
| | 打丢没进 | `cute` | 40%（没中则 25% 补 `chat`） |
| | 席恩自己犯规 | `on_bot_foul` | 100% |
| **终局** | 席恩赢 | `on_win` | — |
| | 席恩输 | `on_lose` | — |
| **换手空档** | 打丢换手且行为语音没命中 | `chat` | 25% |

## taunt — 玩家打丢（4 条）

2. 这一杆很有勇气。
3. 看起来力度不够呢。
4. 哎呀，您这是在故意示弱吗？

## praise — 玩家进球 / 打出好杆（4 条）

1. 这球打得漂亮，看来是要乘胜追击了？
2. 这力道恰到好处，您是不是偷偷练过？
3. 这一球，我没想到能这样打诶。
4. 这可真是意外之喜呢。

## praise_streak — 玩家连续进球（≥2 颗，1 条）
